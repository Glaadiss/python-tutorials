# Conda

## Starting env

```
conda create --name py310 python=3.10
```

```
conda install requests
```

```
conda list
```

```
conda list --explicit > py310.txt
```

```
conda list --revisions
```

```
conda env remove --name py310
```

```
conda env create --file py310.txt 
```

```
pipenv graph
```
